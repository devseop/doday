# Do-Day

## Project Summary

- 해야 하는 일들이 하루에 끝낼 수 없는 것들도 있는데 기존의 `To-do List`는 당일에만 완료되어야 하는걸까?
- 완료되어야 하는 시점을 정해놓고 유연하게 처리해낼 수 있는 `To-do List`가 있으면 어떨까?

**오늘 하루가 기준이 아닌 D-Day를 기준으로 `To-Do`를 계획하고 실행해보자**는 생각으로 시작한 토이 프로젝트입니다.

## Tech Stack

`React` `TypeScript` `Redux Toolkit` `Tailwind CSS`

## Commit Convention

[Gitmoji](https://github.com/carloscuesta/gitmoji)를 사용하여 커밋의 목적이나 의도를 빠르게 식별할 수 있도록 했습니다.
