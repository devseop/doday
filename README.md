# Do-Day

## Project Summary

- 왜 `To-Do List`는 당일에만 완료되어야 하는걸까?
- 하루에 끝나는 것이 아닌 완료되어야 하는 시점을 정해놓고 유연하게 처리해낼 수 있는 `To-Do List`가 있으면 어떨까?

**오늘 하루가 기준이 아닌 D-Day를 기준으로 `To-Do List`를 만들어보자**는 생각으로 시작한 토이 프로젝트입니다.

## Tech Stack

`React` `TypeScript`
